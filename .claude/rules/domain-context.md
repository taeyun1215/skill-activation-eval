# Maum Domain Context

## 서비스 개요

**Maum(마음)** 은 음성 통화 기반 소셜/데이팅 앱입니다.

### 핵심 기능
- **음성 통화**: 1:1 랜덤 매칭 통화, 통화 후 평가(매너 점수)
- **프로필**: 닉네임, 나이, 음성 소개, 사진
- **소셜**: 팔로우, 피드(Moment/Post), 좋아요, 댓글
- **채팅**: 1:1 채팅, 번역, 이미지/음성 메시지
- **그룹톡**: WebSocket 기반 실시간 그룹 대화
- **결제**: 풍선(인앱 화폐), 구독, 인앱 결제

### 도메인 용어

| 용어 | 설명 |
|------|------|
| TalkMate | 상대방 (통화/채팅 상대) |
| Balloon | 인앱 화폐 (풍선) |
| Manner Score | 통화 후 평가 점수 |
| Moment | 짧은 음성/텍스트 피드 |
| Post | 사진/텍스트 게시물 |
| Discovery | 그룹톡 탐색/참여 기능 |
| Vibe | 관심사/분위기 태그 |

### 사용자 흐름

```
가입 → 프로필 설정 → 랜덤 매칭 통화 → 평가 → 팔로우/채팅
                   → 피드 작성/탐색
                   → 그룹톡 참여
                   → 풍선 충전/사용
```

### 비즈니스 규칙 (요약)
- 신고 3회 누적 → 자동 정지
- 통화 최소 30초 이상 시 평가 가능
- 풍선 선물은 친구(팔로우 상호) 관계에서만 가능
- 상세 정책은 `.claude/ai-context/policies.md` 참조
